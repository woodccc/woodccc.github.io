字体

  • font-family
  • line-height
  • color
  • font-family

  1. 对应系统 UI 字体

    • -apple-system => Mac OS X 和 iOS San 下 Safari 里的Francisco 字体
    • BlinkMacSystemFont => Mac OS X 下的 Chrome

  2. 对应一些其他常见系统

    • Segoe UI => Windows 和 Windows Phone
    • Roboto => Android 和 newer Chrome OS’

  3. 备用字体

  • 行高公式: 中文行高 = 字号 x 1.5倍（~ 1.8）， 英文 = 字号 x 1.2倍

  • 字体颜色

  • 精确描述

色彩

  • 色板
  • 色彩应用

色板

  • 色板由 10 个由浅至深的色彩单元格组成







色彩应用

  • 品牌色应用
  • 中性色应用
  • 功能色应用
  • 视觉层次
  • 色彩的易识别性

品牌色

  • 体现产品特性,传播理念

  • 不需要与 logo 完全相同

中性色

  • 关键内容的衬托,功能的引导

功能色

  • 功能信息,状态信息

视觉层次

  • 品牌色赋予在重要信息上、衬以大面积的中性色
  • 界面用色建议不超过三种

色彩的易识别性

  • 色彩更容易被色盲用户识别

色彩对比值校验工具

操作类的色彩搭配都应满足颜色对比值 3:1 的最低标准

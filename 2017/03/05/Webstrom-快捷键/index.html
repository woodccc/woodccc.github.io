Webstrom 快捷键

记录一下常用到的webstrom 快捷键。

  • cmd + B 跳转到方法
  • cmd + 9 打开版本控制，git
  • cmd + 1 打开侧边栏
  • cmd + 7 侧边栏打开函数列表
  • cmd + e 最近文件
  • cmd + shift + o 搜索文件
  • cmd + shift + f 全局搜索
  • cmd + ctrl + s —-Scroll from source 插件，快速在目录中定位当前文件
  • ctrl + g 选中光标处相邻变量
